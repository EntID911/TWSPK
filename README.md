# TWSPK

### 如何使用

```bash
npm install
npm start
```

打开 `http://localhost:3000`



### 核心特性

- 搜索或链接导入 ReaLab 耳机数据
- 重渲染并对比频响、失真、降噪曲线
- 按 octave 频带统计降噪深度
- 导出 PNG、JPG、SVG、WEBP 图片

### 鸣谢

[ReaLab阅真实验室](https://www.realab.com/) 提供专业耳机测试数据支持。
